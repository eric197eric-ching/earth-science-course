# 北市大暑期地球科學課程學習指引

> **課程名稱**：北市大暑期地球科學課程計劃  
> **主題**：地震學、地球物理學 + AI/Agent 輔助教學實作  
> **對象**：中小學現職教師  
> **時程**：2026/7/17 起，每週四上午，共 6 週（每次 4 小時）  
> **目標**：建立地震學基礎、學會使用真實地震資料分析、完成一份可帶回學校的教學作品

---

## 📌 學習前的準備清單

### ✅ 硬體需求
- [ ] **筆電/桌機**：建議配備 Windows 10+ 或 macOS 10.15+，至少 8GB RAM
- [ ] **手機**：支援地震感測的智慧型手機（iOS/Android），用於 Week 2 的地震觀測活動
  - iOS：內建加速度感測器（iPhone 6 以上即可）
  - Android：需支援 **加速度感測器（Accelerometer）**
- [ ] **USB 感測器（可選）**：若學校有 USB 地震感測器（如 [Raspberry Shake](https://raspberryshake.org/) 或 [AS-1](https://www.iris.edu/hq/as-1)），可帶來課堂使用
- [ ] **網路**：穩定的 Wi-Fi 連線（課堂中需下載地震資料、使用雲端工具）

### ✅ 軟體需求
- [ ] **瀏覽器**：Chrome、Edge 或 Firefox（最新版本）
- [ ] **Git**：[下載安裝](https://git-scm.com/downloads)（用於版本控制）
- [ ] **VS Code**：[下載安裝](https://code.visualstudio.com/)（用於編輯程式碼）
- [ ] **Python 3.10+**：[下載安裝](https://www.python.org/downloads/)（用於資料分析）
  - 建議安裝 `pandas`、`numpy`、`matplotlib` 等套件（後續會用到）
- [ ] **Google 帳號**：用於 Google Drive、Google Sheets、Google Form、GAS（Google Apps Script）
- [ ] **GitHub 帳號**：[註冊](https://github.com/join)（用於儲存程式碼和專案）
- [ ] **Hugging Face 帳號**：[註冊](https://huggingface.co/join)（用於 AI 模型和資料集）
- [ ] **Vercel 帳號**：[註冊](https://vercel.com/signup)（用於部署網頁）

### ✅ 先修知識（非必須，但有幫助）
- 基本的 **地球科學概念**（地震、板塊構造、地震波）
- 基本的 **電腦操作**（檔案管理、瀏覽器使用）
- 基本的 **Excel/Google Sheets** 使用經驗（資料分析）

### 📅 Week 2：地震波、加速度、地震與震度
**日期**：2026/7/24  
**目標**：學會使用簡易工具進行地震觀測，理解地震波和震度的概念

#### 🎯 重點內容
1. 地震波的基本概念（P 波、S 波、面波）
2. 加速度與地震強度的關係
3. 震度的定義和計算方法
4. 簡易地震觀測工具介紹（手機、感測器、筆電）

#### 📝 學習步驟
1. **理論學習**：
   - 閱讀 [地震波介紹](https://www.iris.edu/hq/inclass/fact-sheet/seismic_waves) 
   - 觀看 [地震波動畫](https://www.iris.edu/hq/inclass/animation/seismic_waves_and_earths_interior) 
2. **工具準備**：
   - **手機**：安裝地震感測 APP（如 [MyShake](https://myshake.berkeley.edu/) 或 [Seismo](https://play.google.com/store/apps/details?id=com.artemisin.seismo)）
   - **筆電**：測試內建加速度感測器（部分筆電有內建）
   - **USB 感測器**：若有，連接並測試
3. **實作練習**：
   - 使用手機或感測器錄製「敲擊桌面」的振動（模擬地震）
   - 使用 [Phyphox](https://phyphox.org/)（手機物理實驗 APP）錄製加速度資料
4. **資料分析**：
   - 將錄製的資料匯出為 CSV 檔案
   - 在 Google Sheets 中繪製加速度-時間圖

#### 🏆 作業 2：完成簡易地震觀測活動
- **內容**：
  - [ ] 一張手機或感測器地震觀測截圖（顯示振動波形）
  - [ ] 一張 Google Sheets 圖表（顯示加速度-時間變化）
  - [ ] 一份 200 字教師解說（解釋如何用這個活動教學）
  - [ ] 3 題學生問題（用於引導學生思考地震波）
- **提交方式**：上傳至 Google Drive 或 GitHub
- **通過標準**：至少完成 3 項（包含前 2 項）

#### 💡 學習建議
- **新手**：使用手機 APP（如 MyShake）錄製振動，這是最簡單的方法
- **進階**：嘗試用 Python 分析加速度資料（使用 `pandas` 和 `matplotlib`）
- **工具**：
  - [Phyphox 官方教學](https://phyphox.org/experiments/)
  - [MyShake 使用說明](https://myshake.berkeley.edu/docs/)
  - [Google Sheets 圖表教學](https://support.google.com/docs/answer/91362?hl=zh-TW)

### 📅 Week 3：P-S 波到時差、旅行時間曲線與震央定位
**日期**：2026/7/31  
**目標**：學會使用 P-S 波到時差推估震央位置

#### 🎯 重點內容
1. P 波和 S 波的傳播速度差異
2. 旅行時間曲線（Travel Time Curve）的概念
3. P-S 波到時差的計算方法
4. 震央定位的基本原理（三角測量）

#### 📝 學習步驟
1. **理論學習**：
   - 閱讀 [P-S 波到時差介紹](https://www.iris.edu/hq/inclass/fact-sheet/how_do_we_know_where_an_earthquake_originated) 
   - 觀看 [震央定位動畫](https://www.iris.edu/hq/inclass/animation/locating_an_earthquake) 
2. **資料下載**：
   - 從 [IRIS](https://www.iris.edu/hq/retm) 下載 **1 個地震站的 3 分量地震波形資料**（如 [2016 年高雄美濃地震](https://www.iris.edu/hq/retm/event/1456)）
3. **實作練習**：
   - 使用 [Audacity](https://www.audacityteam.org/) 或 [ObsPy](https://docs.obspy.org/) 打開地震波形資料
   - 手動標記 P 波和 S 波的到時
   - 計算 P-S 波到時差
4. **震央定位**：
   - 使用 [IRIS 的震央定位工具](https://www.iris.edu/hq/retm/software-and-services) 
   - 或手動使用旅行時間曲線推估震央距離

#### 🏆 作業 3：完成震央定位教學範例
- **內容**：
  - [ ] 3 個地震站的 P 波和 S 波到時表格
  - [ ] P-S 波到時差計算表
  - [ ] 震央位置推估結果（可用 Google Maps 標記）
  - [ ] GitHub 或 Google Sheets 連結（包含上述資料）
  - [ ] 一份教師提問單（3-5 題問題，用於引導學生）
- **提交方式**：上傳至 GitHub 或 Google Drive
- **通過標準**：至少完成 4 項（包含前 3 項）

#### 💡 學習建議
- **新手**：使用 IRIS 的線上工具（如 [JBR](https://www.iris.edu/hq/retm/software-and-services/jbr)）進行震央定位
- **進階**：用 Python + ObsPy 寫一個簡單的震央定位程式
- **工具**：
  - [IRIS JBR 工具](https://www.iris.edu/hq/retm/software-and-services/jbr)
  - [ObsPy 入門教學](https://docs.obspy.org/tutorial/code_snippets/first-steps.html)
  - [旅行時間曲線計算器](https://www.iris.edu/hq/retm/software-and-services/travel_time_calculator)

### 📅 Week 4：課本第 4 章精讀與真實地震波形資料
**日期**：2026/8/07  
**目標**：深入理解地震波形資料，學會使用專業工具分析

#### 🎯 重點內容
1. 課本第 4 章重點整理（地震波形、頻譜分析、地震能量）
2. 真實地震波形資料的特徵
3. 地震波形分析工具介紹（jAmaSeis、ObsPy、Sac）

#### 📝 學習步驟
1. **閱讀課本**：
   - 精讀課本第 4 章（重點：地震波形、頻譜、能量）
   - 做筆記：整理 10 個重點名詞和概念
2. **資料下載**：
   - 從 [IRIS](https://www.iris.edu/hq/retm) 或 [USGS](https://earthquake.usgs.gov/) 下載 **3 個不同地震的波形資料**
3. **工具安裝**：
   - 安裝 [jAmaSeis](https://www.iris.edu/hq/retm/software-and-services/jamaseis)（Windows）
   - 或安裝 [Sac](https://ds.iris.edu/files/sac/)（Linux/macOS）
4. **實作練習**：
   - 使用 jAmaSeis 打開地震波形資料
   - 進行頻譜分析（FFT）
   - 比較不同地震的波形特徵

#### 🏆 作業 4：完成第 4 章導讀與波形分析小報告
- **內容**：
  - [ ] 一段 200 字的第 4 章導讀（用自己的話解釋重點）
  - [ ] 一張真實地震波形圖（可用 jAmaSeis 或 ObsPy 產生）
  - [ ] 一張頻譜分析圖（FFT 結果）
  - [ ] 3 個地震波形特徵比較（如振幅、頻率、持續時間）
  - [ ] 200 字教學應用說明（如何在課堂上使用這些資料）
- **提交方式**：上傳至 GitHub 或 Google Drive
- **通過標準**：至少完成 4 項（包含前 3 項）

#### 💡 學習建議
- **新手**：使用 jAmaSeis（圖形化介面，易於上手）
- **進階**：用 Python + ObsPy 寫一個波形分析腳本
- **工具**：
  - [jAmaSeis 使用教學](https://www.iris.edu/hq/retm/software-and-services/jamaseis)
  - [ObsPy 波形分析教學](https://docs.obspy.org/tutorial/code_snippets/seismology.html)

## 🎯 學習方法建議

### 📌 一般建議
1. **提前預習**：每週課前花 1-2 小時閱讀相關資料，了解基本概念
2. **積極實作**：課堂中盡量跟著講師操作，不要只聽不做
3. **做筆記**：記錄重點概念、工具使用方法、和自己的疑問
4. **互相討論**：與其他學員交流，分享經驗和心得
5. **及時完成作業**：每週作業盡量在課後 1-2 天內完成，避免累積

3. **詢問講師**：課堂中或課後透過郵件詢問
4. **尋求同學幫助**：其他學員可能也遇過類似的問題

## 💡 額外建議

### 📌 如何讓學習更有效？
1. **設定明確目標**：每週開始前，明確這週要學會什麼
2. **分解任務**：將大任務分解成小步驟，逐步完成
3. **定期回顧**：每週末花 30 分鐘回顧這週的學習內容
4. **實際應用**：嘗試在自己的教學中應用所學的內容
5. **保持好奇心**：多提問、多探索，不要只停留在表面

### 📌 如何應用在教學上？
1. **設計互動活動**：讓學生親自操作地震觀測工具
2. **結合真實案例**：使用真實的地震資料，讓學生感受地震的真實性
3. **培養科學思維**：引導學生提出問題、設計實驗、分析資料
4. **跨領域整合**：結合數學（波形分析）、物理（地震波）、地理（板塊構造）等科目
5. **防災教育**：將地震知識轉化為防災能力，提升學生的安全意識
